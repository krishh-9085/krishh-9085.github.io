# PP Neue Montreal Font Files

Place your licensed font files here with these exact names:

- `PPNeueMontreal-Regular.woff2` (or `.woff`)
- `PPNeueMontreal-Medium.woff2` (or `.woff`)
- `PPNeueMontreal-Bold.woff2` (or `.woff`)

The app is already configured in `src/app/globals.css` to load these files.
